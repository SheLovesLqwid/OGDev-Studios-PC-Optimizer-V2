## Overview

OGDev Studios PC Optimizer is an advanced configuration utility designed to enhance your privacy and security on Windows. This tool is highly recommended for use after a fresh installation of Windows to achieve maximum privacy and security benefits. Depending on your Windows version, Optimizer can also help you apply specific system tweaks.

## How to Disable Defender in Windows 10 1903 and Later

- Restart in SAFE-MODE and run OGDev Studios PC Optimizer with `/disabledefender` switch
- OR -
- Execute OGDev Studios PC Optimizer with `/restart=disabledefender` switch for automated disabling
